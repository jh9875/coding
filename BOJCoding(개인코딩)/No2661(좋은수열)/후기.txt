일단 제일 먼저 생각난건 같은 패턴이 있지 않을까 생각했다.
근데 다시 생각해보니 패턴이 있다는것 자체가 좋은수열이 아니므로 생각을 접었다.

그 다음 생각난건 완전탐색 해보는 경운데 좋운수열중 제일 작은수를 출력하는거기 때문에
자릿수를 만족하는 좋은수열을 하나 찾기만 하면 출력하고 종료하면 됐다.

내가 생각해낸 방법은 각 자릿수마다 1~3까지의 숫자를 넣어보고
좋은수열이면 뒤에 한자리를 더 붙이는것이다.
N자리를 충족하면 그만 붙인다.

이런식으로 하니까 성공이 떴다.

여기서 다른사람들과 프로그램 시간을 비교해보니까 낮게 나온사람들이 있어서 찾아봤다.

전체적인 방식은 비슷한데 좋은수열인지 확인하는 방법이 좀 달랐다.

내가 처음 생각한 방법은 
시작 index가 있고 이걸 점점 뒤로 이동하면서
길이가 len인 두개의 문자열을 만들어서 같은지 비교하는 것이었다.

그리고 빠르게 나온사람의 코드를 보면

for(int i =1; i <=len/2; i++)
{
	String s1 =seq.substring(len-i-i,len-i);
	String s2 =seq.substring(len-i,len);
			
	if(s1.equals(s2))
		return false;
}

이런식으로 나눴는데 시간이 더 빠르게 나왔다.
