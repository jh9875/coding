늘 풀던 bfs문제는 2차원이었다.

이 문제는 3차원으로 탐색하는 문제다..

처음엔 문제를 이해 못해서 오래 읽었지만 늘 풀던 문제랑 비슷한 유형의 문제였다.

2차원의 상,하,좌,우 에다가 3차원의 위,아래를 추가해주면 된다.

원래 2차원에서 4가지 방향을 탐색할땐

int[] dirY ={1, 0, -1, 0};
int[] dirX ={0, 1, 0, -1};

이렇게 했었는데 3차원은 다음과 같이 하면 된다.

int[] dirH ={0, 0, 0, 0, 1, -1};
int[] dirY ={1, 0, -1, 0, 0, 0};
int[] dirX ={0, 1, 0, -1, 0, 0};
